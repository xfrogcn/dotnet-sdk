# 百度AI开放平台 .Net Core SDK

**支持平台：.NET Core 2.0 及以上版本**

**目录结构**

        AipSdk.Baidu.Aip
        ├── Ocr                              //文字识别
        ├── Face                             //人脸识别
        ├── Nlp                              //语言处理基础技术
        ├── ImageCensor                      //图像审核
        ├── ImageClassify                    //图像识别
        ├── Kg                               //知识图谱
        └── Speech                           //语音合成&语音识别


**使用NuGet管理**

在NuGet中搜索 `dotnet.AipSdk`，安装最新版即可。

地址 https://www.nuget.org/packages/dotnet.AipSdk/1.0.0-beta.7.12.15

```
Install-Package dotnet.AipSdk -Version 1.0.0-beta.7.12.15
```
